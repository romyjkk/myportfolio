/* introduction to the project */

  • Dress up game

  • School project

  • HTML - CSS - JavaScript

  • Visual Studio Code - Adobe Illustrator

  • For the subject ‘inleiding programmeren’, which was all about learning the basics of JavaScript, I made a dress up game where you find yourself looking at a character and being able to customize their entire outfit, from skintone to shoes.

  • During this project, I quickly made the decision to make a dress up game, since I’m a fan of fashion and wanted to be able to make a game like this in my own style.

  • All the visual elements are made in Adobe Illustrator, where I mostly used the pencil tool to create everything. Once all those elements were ready, I put them all in an HTML file and started positioning everything in CSS so it would all go into the right spot and I added hover states to make it all a little more smooth.

  • Once I was happy with all the placement of all the elements, it was time for the most challenging part: the JavaScript. This was the first time I was using JavaScript, so it took me a long time to figure out how to make sure the elements appeared and dissapeared when I specifically wanted them to.

  • I first made all necessary variables, for the elements and the buttons, and then continued on to make different functions and if/if else/else if statements.

  • After this I made functions so the elements would be declared. Then I added eventListeners to all the different buttons, so the elements would actually change.

  • Since there were so many elements in the document, it took me a while before everything was working. But once it finally worked, I was very happy. The basics were done!

  • My next step was to add a few more if/else statements. When designing the clothes, I made three different outfits. When you click on the right clothes for one of the outfits, I wanted there to be a background change, and a sound. I made some more variables for the different backgrounds and the sound, and then added the functions with the if/else statements in them.

  • When all this was done, my dress up game was done! I was, and still am, very proud of this. I’m happy with both the visual elements and the code. It was a challenge to make everything work, but in the end I did it!
